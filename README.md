# capable_adaptation_mitigation
Repository of code used in the analysis of the manuscript 'Public support for climate mitigation and adaptation policies across Europe' by Sanford, Mlakar, Smith, et al. and available as a pre-print [here](https://osf.io/preprints/socarxiv/cpwxt_v1).

The data was collected as described in the pre-registration files on OSF [here](https://osf.io/9dmvh/) and [here](https://osf.io/9dmvh/files/sj9a3?view_only=11927d46c8e54fa69f49e16fd3497400).

The data is currently available upon request and will be made publicly available upon publication.
